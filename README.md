# Bad-Ass Geo-Metric Stripe-Patterns :fire::gun::unicorn::sunglasses:

Create awesome geometric patterns using triangles and some clever canvas manipulation

# Install
```shell
$ git clone https://github.com/8-uh/stripe-patterns
$ cd stripe-patterns
$ npm install
$ npm start
```
